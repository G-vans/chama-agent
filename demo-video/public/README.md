# Recording assets

Export these clips into this directory, ideally at 1920×1080:

- `dashboard.mp4`
- `health-report.mp4`
- `chat-intelligence.mp4`
- `payment.mp4`
- `statement.mp4`
- `codex.mp4`
- `voiceover.wav`

Then replace the corresponding `null` values in `src/config.ts` with the filenames.

The final composition is 2:59 at 30 fps. Clips are automatically cropped to fill the frame, with branded explanatory overlays added by Remotion.
