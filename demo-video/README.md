# Chama Agent demo video

This Remotion composition produces the under-three-minute OpenAI Build Week demo.

```sh
npm install
npm run studio
```

Record the product clips listed in `public/README.md`, add them to `public/`, and update `src/config.ts`. Preview timing in Remotion Studio, then render:

```sh
npm run render
```

The output is written to `out/chama-agent-demo.mp4`.
